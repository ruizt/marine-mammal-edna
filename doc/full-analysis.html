Predicting whale density from eDNA: full analysis

This vignette presents results for the complete analysis across all three eDNA markers (16S rRNA, 18S rRNA V4, 18S rRNA V9) and three baleen whale species (blue whale BM, fin whale BP, humpback whale MN) — nine (marker × species) model combinations in total. For a step-by-step walkthrough of the pipeline, see vignette("single-iteration").

Requires: processed data files in data/. Run source("analysis/setup-data.R") to download from Zenodo (v1.1.0; ~16 MB core files). Pre-computed results (stable sets, fitted models) are loaded from rslt/ rather than recomputed; full reproduction takes approximately 3 hours per marker via analysis/run-all.R.

library(tidyverse)
library(here)

Study system and data

The study uses eDNA collected at CalCOFI stations in the southern California Current Ecosystem across 25 quarterly cruises (2014–2020). At each cruise, water samples were filtered and amplified with three eDNA markers targeting different components of the microbial and zooplankton community. Simultaneously, blue, fin, and humpback whale density was estimated from visual line-transect surveys (see vignette("line-transect")).

Stability selection results

For each (marker, species) combination, stability selection was run across a grid of 55 eta × 9 ncomp = 495 hyperparameter configurations and 25 LOO subsamples. The best stable set was then chosen by nested cross-validation (see analysis/stability-selection-*.R and analysis/nested-validation-*.R).

ss_16s   <- read_rds(here("rslt/stability-selection/16s/stable-sets.rds"))
ss_18sv4 <- read_rds(here("rslt/stability-selection/18sv4/stable-sets.rds"))
ss_18sv9 <- read_rds(here("rslt/stability-selection/18sv9/stable-sets.rds"))

stable_sets_all <- bind_rows(
  mutate(ss_16s,   marker = "16S"),
  mutate(ss_18sv4, marker = "18S V4"),
  mutate(ss_18sv9, marker = "18S V9")
) |>
  mutate(n.asv = map_int(ss, length),
         species = recode(species, bm = "Blue", bp = "Fin", mn = "Humpback"))

The number of stable ASVs varies considerably across marker–species combinations, reflecting both the different community targets of each marker and the varying predictability of each species:

Model fitting and metrics

Full-data PLS models were fit to each stable set. Results are loaded from rslt/models/:

load(here("rslt/models/fitted-models-18sv4.RData"))

load(here("rslt/models/fitted-models-18sv9.RData"))

all_metrics <- bind_rows(
  mutate(metrics_16s,   marker = "16S"),
  mutate(metrics_18sv4, marker = "18S V4"),
  mutate(metrics_18sv9, marker = "18S V9")

Comparison with naive baselines

A key question is whether eDNA models do better than simple baselines that ignore eDNA entirely. Two baselines were computed for each species:

  • Lag: the density estimate from the previous cruise (autoregressive baseline)
  • Seasonal mean: the mean density for the same calendar quarter across all years
naive_preds <- read_rds(here("rslt/models/naive-preds.rds"))

# Best eDNA RMSPE per species across markers
best_edna <- all_metrics |>
  group_by(Species) |>
  slice_min(`RMSPE (density)`) |>
  select(Species, Marker, `RMSPE (density)`)

naive_preds |>
  mutate(Species = recode(species, bm = "Blue", bp = "Fin", mn = "Humpback")) |>
  pivot_longer(c(rmspe.lag, rmspe.mean),
               names_to = "Baseline", values_to = "RMSPE",
               names_prefix = "rmspe.") |>
  mutate(Baseline = recode(Baseline, lag = "Lag", mean = "Seasonal mean")) |>
  left_join(rename(best_edna, `eDNA RMSPE` = `RMSPE (density)`),
            by = "Species") |>
  pivot_longer(c(RMSPE, `eDNA RMSPE`),
               names_to = "Model", values_to = "rmspe") |>
  mutate(Model = if_else(Model == "RMSPE", Baseline, paste0("eDNA (", Marker, ")"))) |>
  ggplot(aes(x = Species, y = rmspe, fill = Model)) +
  geom_col(position = "dodge") +
  labs(y = "RMSPE (ind. km\u207b\u00b2)", fill = NULL) +
  scale_fill_brewer(palette = "Set2")
RMSPE for the eDNA models (best marker per species) vs. naive baselines. Lower is better.

The eDNA models substantially outperform both baselines for fin whale (BP) and show competitive performance for blue (BM) and humpback (MN) whale.
